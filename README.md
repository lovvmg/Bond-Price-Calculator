# Bond-Price-Calculator
A simple, user-friendly Desktop Application built with Python and Tkinter to calculate the Price of a Bond. This tool accounts for coupon rates, yield rates, redemption values, and various payment frequencies.
# Features
Dynamic Calculation: Calculates bond price based on Face Value, Yield Rate, Coupon Rate, and Redemption Value.
Flexible Frequencies: Supports multiple payment schedules:
  Annually (m = 1)
  Semiannually (m=2)
  Quarterly (m=4)
  Monthly (m=12)
Interactive UI: Built with tkinter.ttk for a clean, native look.
Instant Results: Displays the calculated price in a pop-up message box.
# Requirements
Python 3.x
Tkinter (usually comes pre-installed with Python)
